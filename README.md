# Radio tracks to Spotify
This folder contains N8N workflow `.json` files that scrape currently playing songs from selected online radio stations and add them to a Spotify playlist. The playlist is cleared every day, and new tracks are added. The workflow adds new songs every few minutes.

At the moment, there are 3 workflows, each for a different radio station:
- Radio Kampus (Polish)
- Q-Music channel (Dutch) (thanks to [g4ntel](https://github.com/g4ntel))
- Studio Brussel (Belgian) (thanks to [g4ntel](https://github.com/g4ntel))
